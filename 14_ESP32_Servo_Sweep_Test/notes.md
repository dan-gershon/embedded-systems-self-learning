// ============================================================
// Experiment 14 – ESP32 Servo Sweep Test
//
// This project controls an SG90 servo motor using an ESP32.
// The servo moves between several fixed angles in order to test
// basic mechanical actuator control.
//
// Main concepts:
// - Servo motor control
// - PWM signal for servo positioning
// - 50 Hz servo control signal
// - Pulse width and angle relationship
// - Basic actuator testing
// - ESP32Servo library
//
// Hardware:
// - ESP32 Development Board
// - SG90 Servo Motor
//
// Connections:
// - Servo Brown / Black  -> GND
// - Servo Red            -> 5V / VIN
// - Servo Orange / Yellow -> GPIO18
//
