# Embedded Systems Self-Learning Lab

A growing hands-on embedded systems portfolio focused on microcontrollers, digital communication, signal measurement, control, and electronics experiments.

This repository documents my practical learning process in embedded systems and electrical engineering through small, measurable experiments.  
The goal is to understand not only how to write firmware, but also how the code becomes real electrical signals that can be measured, decoded, and analyzed.

## Goals

- Build practical experience with microcontrollers.
- Learn how to control and measure digital signals.
- Use lab tools such as a USB logic analyzer.
- Understand communication protocols from the signal level.
- Practice clean firmware structure and documentation.
- Create a portfolio of hands-on electronics and embedded systems experiments.

## Hardware

- ESP32 development board
- USB logic analyzer
- Dupont wires
- Additional sensors, modules, and circuits will be added over time

## Software Tools

- Arduino IDE
- PulseView / sigrok
- GitHub
